# 문법 설명

# '#'으로 시작하는 줄은 주석이며 무시됩니다.

# "${오디오 파일}"은 그 다음 '$'가 오기 전까지의 키워드 연산들을 해당 "오디오 파일"에 연결시킨다는 뜻입니다.
# 키워드 연산의 최종 결과가 참이 되면 해당 오디오 파일이 선택됩니다.
# 한 메세지에 대해 여러 오디오 파일이 선택되었다면 가장 아랫줄에 선언한 파일이 재생됩니다.

# 키워드 연산을 선언하는 방법은 "{연산}{키워드}"입니다.
# 연산의 종류는 아래와 같으며 위에서 아래로 실행됩니다.
# '*' : 키워드가 메세지에 있고 이전 연산의 결과가 참이라면 참을, 그 외엔 거짓을 반환.
# '+' : 이전 연산 결과가 참이면 참을, 아니라면 키워드가 메세지에 있는지 여부를 반환.
# '~' : 키워드가 메세지에 있다면 거짓을, 그 외엔 이전 연산 결과를 반환.
# '!' : 키워드가 메세지에 있다면 참을, 그 외엔 이전 연산 결과를 반환.
# 첫 연산이거나 이전 연산의 결과가 없다면 '*'일 경우엔 참, '+'일 경우엔 거짓을 이전 연산 결과로 사용합니다.

# 예시 1)
# 메세지 : "포항 미소지진"
# 코드 :
# $test.mp3
# *지진
# *부산
# 결과 :
# 일단 "지진"이 있으므로 참이 반환되고 이어서 "부산"은 없으므로 거짓이 반환됩니다.

# 예시 2)
# 메세지 : "포항 미소지진"
# 코드 :
# $test.mp3
# *지진
# +부산
# *포항
# 결과 :
# 일단 "지진"이 있으므로 참이 반환되고 이어서 "부산"은 없지만 이전 결과가 참이므로 참이 반환됩니다.
# 이어서 "포항"이 있으므로 최종 결과는 참이 됩니다.

# 예시 3)
# 메세지 : "일본 오사카 해역 규모 5.4 지진"
# 코드 :
# $test.mp3
# *지진
# *해역
# ~일본
# 결과 :
# 일단 "지진"이 있으므로 참이 반환되고 이어서 "해역"도 있으므로 참,
# 하지만 "일본"이 있으므로 최종 결과는 거짓이 됩니다.
# 일본 외의 지역에서 발생한 해역 지진을 인식할 수 있는 조건이 되겠죠.

$normal.mp3
*Normal Level

$high.mp3
*High Level

$critical.mp3
*Critical Level
